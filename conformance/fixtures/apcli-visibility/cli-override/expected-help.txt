Usage: apcore-cli [options] [command]

apcore CLI — execute apcore modules from the command line

Options:
  --version            Show apcore-cli version
  --log-level <level>  Logging level (DEBUG|INFO|WARNING|ERROR) (default:
                       "WARNING")
  --verbose            Show all options in help output (including built-in
                       apcore options)
  -h, --help           display help for command

Commands:
  apcli                apcore-cli built-in commands
  help [command]       display help for command
